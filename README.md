# Curriculum — Jessica Mayra Quispe Rico

Sitio web personal tipo CV desarrollado con HTML y CSS.

## Contenido

El sitio presenta la información profesional y académica en las siguientes secciones:

- **About me** — breve descripción personal
- **Experience** — proyectos académicos desarrollados
- **Education** — formación universitaria
- **Skills** — tecnologías y herramientas que manejo
- **Languages** — idiomas

## Tecnologías utilizadas

- HTML5
- CSS3
- Google Fonts (Inter)

## Estructura del proyecto

```
├── index.html
├── styles/
│   ├── base.css       # Estilos globales, footer
│   ├── navbar.css     # Barra de navegación
│   ├── header.css     # Header con foto y datos de contacto
│   └── cards.css      # Cards de proyectos, skills y secciones
└── Imagenes/          # Foto de perfil e íconos
```

## Características

- Diseño responsive (mobile y desktop)
- Navbar con menú hamburguesa en mobile y links horizontales en desktop
- En desktop, el header muestra la foto a la izquierda y los datos de contacto a la derecha
- Efectos hover en links de navegación, redes sociales y botones de proyectos
- Contenido centrado con ancho máximo en pantallas grandes
